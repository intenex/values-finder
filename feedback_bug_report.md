Feedback/Bug report

Device: PC/Chrome

1. Final 10 Values Page  
   * When a user reaches the "10 values" page, if they click the back button or refresh the page to edit even a single answer from the previous question, the system generates a completely different set of values. The new set of values includes some options the user never chose, while actively replacing or adding others randomly.  
   * Expected Behavior: The system should retain the user's previously/consistently selected values.  
2. Assessment progress achievements pop-up  
   * Alignment/Placement: The pop-ups are consistently off-center. Most are slightly askew. A few appeared vertically, and another appeared diagonally.  
     * Recommended: Standardize the alignment so all pop-ups anchor cleanly to the exact center of the screen (or a consistent, non-intrusive corner).  
   * Styling Recommendation: Current progress achievement pop-ups have a high-contrast color scheme that deviates significantly from the site's general theme. This causes them to look like intrusive pop-up ads rather than native platform notifications.  
     * Recommended: Re-skin the notifications to match the site’s core color palette.  
3. Login/Signup flow  
   * Currently, registration happens either during or after the assessment options are completed. If a user gets tired, experiences a browser crash, or wants to pause and continue later, they lose all progress and have to restart all over again.  
     * Recommended: Require login or sign-up at the very beginning of the assessment. This ensures user progress is saved continuously and prevents user frustration.  
     * Successful sign in should cause the exercise to open so people know they’re signed in.   
4. Color for Values selection feedback  
   * When selecting the "Most" (Green) and "Least" (Red) values, the use of stark Red feels overly negative. Because all the values presented are positive/wonderful traits, highlighting the "Least" choice in red makes the user feel like they are making a "wrong" or negative choice.  
     * Recommended: Consider switching to green and blue with a more neutral color palette so the “least” option doesn’t seem negative.
